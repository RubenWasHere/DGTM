What Is Software Engineering?

Anytime you visit a webpage or use an internet-powered application, you’re engaging with the end result of a software engineer’s work! Software engineers are computer professionals who use knowledge of engineering principles and programming languages to create a range of software products, develop computer games, and run network control systems.

According to Absolute IT Salary Report 2021 the salary of a software engineer ranges from a lower quratile of $83k as a Junior Software Engineer to $130k as a Senior Software Engineer.

Some important skills for this career pathway include...
Communication
Teamwork
Computer programming and coding
Problem-solving
Multitasking
Attention to detail

Computer Programming and Coding

This is one of the fundamental soft engineering skills. Computer programming focuses on algorithms, which have become a part of our daily lives. There are a number of computer programming languages, and job requirements can vary depending on the position you’re applying for. Some exmaples of the most in demand and well known languages include:

App Development

Java, one of the most widely used programming languages
is used as the server-side language for most back-end development projects, including those involving big data and Android development. Java is also commonly used for desktop computing, other mobile computing, games, and numerical computing.

General Purpose

Python, is a general purpose coding language which means that, unlike languages like HTML, CSS, and JavaScript, it can be used for other types of programming, such as: back end development, software development, data science, and writing system scripts (automation) among other uses.

Dekstop Applications and Games

C#, is widely used for developing desktop applications, web applications and web services. It is used in creating applications of Microsoft at a large scale. C# is also used in game development in the program Unity.

Web Development

HTML,(HyperText Markup Language) is what is used to create and structure all websites on the internet. It is often paired with the language CSS (Cascading Style Sheets) to style the HTML doccument with various aesthetic changes such as font, color and positioning of images.
